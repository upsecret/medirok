# 실행 가이드 — 디오디피부과의원 청담 구글맵 리뷰 적용

> **이 문서는 Claude Code CLI가 그대로 실행하기 위한 런북입니다.**
> 목표: `scripts/seed-dod.ts`를 실행해 디오디피부과의원 청담의 평점·후기를
> 구글맵 데이터(**★4.8 · 리뷰 45개 · 후기 5건**)로 라이브 반영한다.
> upsert(멱등) 스크립트라 여러 번 실행해도 안전하다.

---

## 0. 사전 조건

- `.env.local`에 `DATABASE_URL`(Neon Postgres) 설정돼 있을 것
- Node 실행환경 정상 (※ `ci-info: vendors.map is not a function` 에러는
  Node 24 + `node --import tsx/esm` 조합 이슈 — Claude Code CLI 환경에서 해결됨)
- 의원은 이미 등록돼 있음 → 이번 실행은 **기존 레코드 update(upsert)**

---

## 1. 실행 (Claude Code CLI)

```bash
# 프로젝트 루트에서
npm run seed:dod
```

`seed:dod` = `node --env-file=.env.local --import tsx/esm scripts/seed-dod.ts`

**기대 출력:**
```
• 지역(regions) upsert
  ✓ update  regions/서울
  ✓ update  regions/강남구
  ✓ update  regions/청담동      (최초 1회는 create)
• 의원(hospitals) upsert — 디오디피부과의원 청담
  ✓ update  hospitals/디오디피부과의원청담
✅ 디오디피부과의원 청담 등록 완료
```

> 만약 `ci-info` 에러가 재발하면(다른 셸 환경):
> `npx tsx --env-file=.env.local scripts/seed-dod.ts` 로 직접 실행하거나,
> Node LTS(22.x)로 전환 후 재실행.

## 2. 검증
